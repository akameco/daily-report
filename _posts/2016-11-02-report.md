雑に日報開くために`.zshrc`に関数書いた。

```sh
daily() {
	cd $GHQ/akameco/daily-report/_posts
	local t=$(date +"%Y-%m-%d")
	local f="$t-report.md"
	if [ -e $f ]; then
		vim $f
	else
		touch $f
		vim $f
	fi
}
```

雑に`daily`と書くと新しい日報をつくる。
